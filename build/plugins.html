Plugins

There are some cool CoffeeLint plugins.

Editors

  • vscode-coffeelint is a plugin for Visual Studio Code;
  • linter-coffeelint is a plugin for Atom;
  • SublimeLinter-coffeelint is a plugin for Sublime Text;
  • vim-coffee-script is a plugin for Vim (supports CoffeeLint);
  • Syntastic is a plugin for Vim (support CoffeeLint);
  • CoffeeLintNode is a plugin for Emacs;
  • flymake-coffee is a plugin for Emacs Flymake.

Build tools

  • grunt-coffeelint is a plugin for Grunt;
  • gulp-coffeelint is a plugin for Gulp;
  • coffeelint-loader is a plugin for Webpack;
  • coffeelint-brunch is a plugin for Brunch.io.

Source Control

  • overcommit is a configurable and extendable Git hook manager (supports CoffeeLint).

Other

  • coffeelint-ruby is a set of bindings for Ruby;
  • codacy-coffeelint is a docker container for Codacy, that runs CoffeeLint;
  • arcanist-coffee-lib is a set of Arcanist extensions, including a wrapper for CoffeeLint;
  • mimosa-coffeelint is a plugin for Mimosa.
