Rules

There are over 40 rules built into CoffeeLint. Also coffeescript_error means an error while source parsing.

Possible errors

Rule name Rule message
cyclomatic_complexity The cyclomatic complexity is too damn high
duplicate_key Duplicate key defined in object or class
indentation Line contains inconsistent indentation
line_endings Line contains incorrect line endings
missing_fat_arrows Used this in a function without a fat arrow =>
missing_parseint_radix parseInt() is missing the radix argument
no_backticks Backticks are forbidden
no_debugger Found debugging code
no_empty_functions Empty function
no_interpolation_in_single_quotes Interpolation in single quoted strings is forbidden
no_nested_string_interpolation Nested string interpolation is forbidden
no_throwing_strings Throwing strings is forbidden

Stylistic issues

Rule name Rule message
arrow_spacing Function arrows (-> and =>) must be spaced properly
braces_spacing Curly braces {} must have the proper spacing
bracket_spacing Square brackets [] must have the proper spacing
camel_case_classes Class name should be in PascalCase style
colon_assignment_spacing Colon : assignment without proper spacing
empty_constructor_needs_parens Invoking a constructor without parens () and without arguments
ensure_comprehensions Comprehensions must have parentheses () around them
eol_last File does not end with a single newline
max_line_length Line exceeds maximum allowed length
newlines_after_classes Wrong count of blank lines between a class and other code
no_empty_param_list Empty parameter list is forbidden
no_implicit_braces Implicit braces {} are forbidden
no_implicit_parens Implicit parens () are forbidden
no_plusplus The increment ++ and decrement -- operators are forbidden
no_private_function_fat_arrows Used the fat arrow => for a private function
no_spaces Line contains space indentation
no_stand_alone_at @ must not be used stand alone
no_tabs Line contains tab indentation
no_this Use @ instead of this
no_trailing_semicolons Line contains a trailing semicolon ;
no_trailing_whitespace Line ends with trailing whitespace
no_unnecessary_double_quotes Unnecessary double quotes "" are forbidden
no_unnecessary_fat_arrows Unnecessary fat arrow =>
non_empty_constructor_needs_parens Invoking a constructor without parens () and with arguments
object_shorthand Use property-value shorthand when using explicit braces {}
prefer_english_operator Don't use &&, ||, ==, !=, or !
prefer_fat_arrows_in_methods Require fat arrows => inside method bodies
prefer_logical_operator Don't use is, isnt, not, and, or, yes, on, no, off
space_operators Operators must be spaced properly
spacing_after_comma a space is required after commas ,
transform_messes_up_line_numbers Transforming source messes up line numbers
